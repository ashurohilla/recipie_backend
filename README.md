# recipie_backend
# steps
## create a environment 
`python -m venv env` 

## activate environment
`cd env` 
`cd Scripts` 
`activate`
## coming out to main dir 
`cd ..`
`cd ..`
## makemigrations
`python manage.py makemigrations`

## migrate 
`python manage.py migrate`

## starting server 
`python manage.py runserver 8002`
